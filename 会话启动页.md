# 会话启动页

这页只保留你真的会复制出去用的模板。

## 最小启动模板

```text
请按当前 wiki 规则工作。

开始前先读：
1. Home.md
2. README-zh.md
3. 快速开始.md
4. 使用手册.md

如果这是某个项目任务，再读：
5. 项目仓库根目录的 wiki.context.json
6. 项目仓库根目录的 AGENTS.md 或 CLAUDE.md

要求：
- 优先使用现有脚本
- 只读取必要页面
- 如果结果有长期价值，写回 wiki
- 如有页面变更，更新日志并重建索引
```

## 项目任务模板

```text
请在完成当前项目任务的同时维护项目 wiki。

开始前先读：
1. Home.md
2. README-zh.md
3. 快速开始.md
4. 使用手册.md
5. 当前项目仓库根目录的 wiki.context.json

要求：
1. 先定位当前项目对应的 wiki 页面
2. 完成代码或分析任务
3. 同步更新受影响的项目页面
4. 如果形成了长期结论，写回项目 wiki 或 40_outputs
5. 最后更新日志并重建索引
```

## 资料摄入模板

```text
请把这份资料摄入当前 wiki，不要只保存原文件。

要求：
1. 使用 ingest_source.py 保存文件并创建来源笔记
2. 提炼核心事实、约束、风险和决策信息
3. 更新真正受影响的页面，而不是只新建孤立页
4. 如果有复用价值，提升到共享层
5. 更新日志并重建索引
```

## 查询沉淀模板

```text
请基于当前 wiki 回答问题，并在结论有长期价值时写回 wiki。

要求：
1. 先用索引和 search_wiki.py 定位页面
2. 只读必要内容
3. 回答时引用具体页面
4. 如果结论稳定，使用 file_back_query.py 写回
```
